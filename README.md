# micro-frontend-demo

## vite-project
主应用，基于react和vite

### 启动
```shell
npm run dev
```
### 打包
```shell
npm run build
```

## vue2-component
基于vue2的子组件

### 打包
```shell
npm run lib
```
### 提供服务
```shell
npm run lib:serve
```
### 开发启动
```shell
npm run serve
```
