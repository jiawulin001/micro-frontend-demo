## vite-project
Main application, based on react and vite

### Project setup
```shell
npm run dev
```
### Compiles
```shell
npm run build
```

## vue2-component
Sub-components based on vue2

### Building component libraries
```shell
npm run lib
```
### Services Provided
```shell
npm run lib:serve
```
### Development Launch
```shell
npm run serve
```
